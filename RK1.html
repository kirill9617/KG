Рубежный контроль №1

1. Написать программу построения отрезка АВ (A(10, 10), B(20+i, 100), где i - номер по журналу)

2. Дан массив а0, а1, а2, ..., an, такой, что (а0 < a1 < а2 < ... < an). Необходимо сгруппировать все элементы массива, идущие подряд. Вывести последовательность элементов в скобках (arr_2*i, arr_(2*i+1)), где arr_2*i - четная (элементы по порядку) группа, arr_(2*i+1) - следующая за ней нечетная (элементы не по порядку) группа. Пример: Вход: [1, 2, 10, 20, 21, 22, 31, 35, 45, 46] Выход: (1, 2, 10)(20, 21, 22, 31)(35, 45, 46)
